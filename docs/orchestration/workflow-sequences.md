# Workflow Sequences

## New Gameplay Feature
1. `/plan`
2. `/gdd` or `/tech-design` as appropriate
3. `/tdd`
4. implementation by the responsible agent/programmer
5. `/verify`
6. `/update-docs`

## Vertical slice
1. `/vertical-slice`
2. `/milestone-plan`
3. design + implementation
4. `/qa-plan`
5. `/playtest-report`
6. `/perf-budget`
7. `/release-check`
