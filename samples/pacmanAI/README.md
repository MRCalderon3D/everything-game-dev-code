# PAC-MAN AI

Classic PAC-MAN arcade game for the browser. Built with vanilla JavaScript and HTML5 Canvas as a `/full-game` command sample from the Everything Game Dev scaffold.

## Genre
Arcade / Maze Chase

## How to Run

```bash
cd samples/pacmanAI
npx serve .
```

Then open the URL shown in the terminal (usually `http://localhost:3000`).

Or use any static file server:

```bash
npx http-server .
# or
python -m http.server
```

## Controls

| Action | Keyboard | Touch |
|--------|----------|-------|
| Move Up | Arrow Up / W | Swipe up |
| Move Down | Arrow Down / S | Swipe down |
| Move Left | Arrow Left / A | Swipe left |
| Move Right | Arrow Right / D | Swipe right |
| Pause | Escape / P | Tap top area |
| Menu Select | Enter / Space | Tap |

## Features

- Classic PAC-MAN maze (28x31 tiles)
- 4 ghosts with scatter/chase/frightened AI (Blinky, Pinky, Inky, Clyde)
- Power pellets with ghost vulnerability
- Level progression (increasing ghost speed)
- Score system with extra lives at 10,000 points
- Procedural audio (Web Audio API)
- Canvas-rendered retro visuals
- High score persistence (localStorage)
- Touch controls for mobile
