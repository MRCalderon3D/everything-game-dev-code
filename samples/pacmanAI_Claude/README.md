# PAC-MAN Claude

A classic-style PAC-MAN arcade game for the browser, built with vanilla JavaScript and HTML5 Canvas. Generated end-to-end by the `/full-game` command from the [Everything Game Dev Code](https://github.com/MRCalderon3D/everything-game-dev-code) scaffold.

## Genre
Arcade / Maze Chase

## How to Run

```bash
cd samples/pacmanAI_Claude
npx serve .
```

Then open the URL shown in the terminal (usually `http://localhost:3000`).

Any static file server works:

```bash
npx http-server .
# or
python -m http.server
```

## Features

- Classic 28×31 maze with horizontal tunnel on row 14
- 4 ghosts with distinct AI (Blinky direct chase, Pinky ambush, Inky flanker, Clyde erratic)
- Scatter / chase / frightened / eaten mode cycle
- 4 power pellets with combo chain (200 / 400 / 800 / 1600 points)
- Lives, level progression, extra life at 10,000 points
- High score persistence via `localStorage`
- Procedural audio (Web Audio API, no external files)
- Touch controls: swipe + on-screen D-pad on mobile
- Pause / resume / menu navigation

## What You See / Hear

- **Visuals:** All sprites are drawn with Canvas 2D primitives — no external images. PAC-MAN is a yellow disc with an animated mouth; each ghost has a unique color and floating eyes; power pellets pulse softly; walls use a layered blue with a darker inner edge.
- **Audio:** Every sound effect is synthesized on demand via the Web Audio API. Pellet pickups alternate between two short chirps; power pellets bloom with a warm sustain; eating a ghost plays an ascending arpeggio; the death sound is a descending dissonant sweep.
