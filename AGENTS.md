# AGENTS.md

Global instructions for compatible harnesses.

## Principles
- Delegate by discipline and by engine.
- Maintain isolation between Unity, Unreal, and Godot.
- Design before implementing.
- Validate gameplay, performance, UX, and stability.
- Keep documentation alive: GDD, technical TDD, QA, release.

## Structure
- `agents/`: especialistas
- `skills/`: how to execute tasks
- `commands/`: fast entry points
- `rules/`: standards
- `docs/orchestration/`: relationships between parts
