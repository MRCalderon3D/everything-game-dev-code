# CONTRIBUTING

## Conventions
- lowercase filenames with hyphens
- one agent per file
- one skill per directory with `SKILL.md`
- rules separated into `common/`, `unity/`, `unreal/`, `godot/`
- do not mix patterns from different engines in the same engine-specific skill

## Contribution Types
- new agents
- new skills
- new commands
- new rules
- hooks
- manifests/profiles
- documentation templates
