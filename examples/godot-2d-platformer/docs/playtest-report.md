# Clockspring Playtest Report

## Session Goal

Check whether first-hour movement mastery and checkpoint recovery feel fair for players who are comfortable with 2D platformers but new to this project.

## What We Learned

- Players understood wall-jump timing after one or two retries
- Checkpoint readability was strong when landmarks were visible
- Fail-state reset felt too slow in the gear-lift room
- Two players missed the optional tutorial prompt for chained dash timing

## Recommended Follow-Up

- Shorten reset timing after death in high-repetition rooms
- Improve contrast and framing for the chained dash tutorial prompt
- Review signal-driven checkpoint reset logic for edge-case desync
