# Unity

Extends `rules/common/` with Unity-specific conventions.

## Layering
- `rules/common/` defines engine-neutral policy.
- `rules/unity/` adds Unity-only implementation rules.
- Unity rules must never prescribe Unreal or Godot patterns.
