# Godot

Extends `rules/common/` with Godot-specific conventions.

## Layering
- `rules/common/` defines engine-neutral policy.
- `rules/godot/` adds Godot-only implementation rules.
- Godot rules must never prescribe Unity or Unreal patterns.
